# HR Real Estate

A modern, premium real estate web application for browsing and managing DHA (Defence Housing Authority) properties in Lahore, Pakistan.

## Features

- **Property Listings** — Browse, search, and filter residential and commercial properties across all DHA phases
- **Admin Dashboard** — Secure admin panel for managing properties, testimonials, and inquiries
- **Responsive Design** — Fully responsive UI optimized for desktop, tablet, and mobile devices
- **SEO Optimized** — Server-side rendered pages with proper metadata and sitemap
